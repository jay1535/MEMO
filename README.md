# 📝 Memo

**Memo** is a simple and intuitive **note-taking app** designed to help users organize their thoughts, tasks, and ideas efficiently.  
With Memo, you can quickly create, edit, and manage notes with ease.

---

### 📚 Topics & Tech Stack

- ⚛️ **React.js** — Frontend UI library
- 🖋️ **Bootstrap 5** — Responsive styling
- 🌳 **Node.js** — Backend runtime
- 🚀 **Express.js** — Server framework
- 🍃 **MongoDB** — NoSQL database

---

### 🚀 Features

- ✅ Create, edit, and delete notes  
- ✅ Responsive and user-friendly UI  
- ✅ Stores notes in MongoDB database  
- ✅ Fast and lightweight backend with Node.js + Express  
- ✅ Styled with Bootstrap 5 for modern look & feel

### 🛠️ Getting Started

### 1️⃣ Clone the repository
```bash
git clone https://github.com/your-username/memo.git
cd memo
```
### 📦 Install Dependencies

To install all dependencies:
```bash
npm install
npm install react
npm install bootstrap
npm install express
npm install mongoose
```
### 📄 `.env.local`

```env

 MONGODB_URI=your_mongodb_connection_string
 PORT=5000
```
### 🚀 Run Development Server

After installing dependencies and setting up your `.env.local`, start the development server:
```bash
npm start

#For server
cd server
npm start
```
